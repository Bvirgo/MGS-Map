# MGS-Map
- [English Manual](./README.md)

## 概述
- Unity制作场景小地图，在地图上绘制运动轨迹插件包。

## 环境
- Unity 5.0 或更高版本。
- .Net Framework 3.0 或更高版本。

## 实现
- GlobalMap：全局地图。
- LocalMap：局部地图。
- MapFlag：地图（静态/动态）标记。
- FlagTrail：地图动态标记运动轨迹。
